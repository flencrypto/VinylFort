# VinylFort

A comprehensive authentication and privacy management system with clear privacy policy and authentication state pages.

## Overview

VinylFort provides a secure authentication system with transparent privacy policies and user-friendly pages for different authentication states.

## Features

- **Comprehensive Privacy Policy**: Detailed privacy policy covering data collection, usage, and user rights
- **Authentication Accepted Page**: Information page for successful authentication
- **Authentication Declined Page**: Helpful page for failed authentication attempts with troubleshooting tips
- **Responsive Design**: Mobile-friendly interface that works on all devices
- **Clean UI**: Modern and professional design with clear visual feedback

## Pages

### Privacy Policy (`pages/privacy-policy.html`)
Comprehensive privacy policy that covers:
- Information collection practices
- Data usage and processing
- Authentication and authorization details
- Data security measures
- User rights under data protection laws
- Contact information

### Authentication Accepted (`pages/auth-accepted.html`)
Success page shown when authentication is accepted:
- Visual confirmation with animated checkmark
- Information about next steps
- Links to dashboard and other resources
- Security notices

### Authentication Declined (`pages/auth-declined.html`)
Information page shown when authentication fails:
- Clear error indication
- Possible reasons for authentication failure
- Action buttons (Try Again, Reset Password, Contact Support)
- Troubleshooting tips
- Privacy information about logging

## Usage

1. Open `index.html` in a web browser to view the main page
2. Navigate to individual pages using the links provided
3. All pages are interconnected for easy navigation

## Viewing the Project

Simply open `index.html` in any modern web browser. The project uses HTML and CSS only, with no build process required.
